# Add "My Journey" Progress Dashboard to Home Page

**Checkpoint before running: "pre-progress-dashboard"**

Add a "My Journey" card to the Home page, positioned between the hero banner and the Stories of Change carousel. See the attached screenshot for the visual design reference.

---

## What it looks like

A white card with rounded corners containing:

**Left side:** A circular progress ring (80px) showing the user's overall completion percentage as a number in the center. The ring fills with dark teal (#34737A). The unfilled portion is light gray (#F1EFEF).

**Right side of the ring:**
- Title: "My Journey" (15px, bold, #302D2E)
- Subtitle: "Five Foundations for Real Change" (12px, #868180)
- Encouraging line below: "Keep going — every step matters" for in-progress users. "All foundations complete" with a green checkmark for users at 100%.

**Below the header row:** Five small circular progress rings in a horizontal row (one per pillar), evenly spaced. Each ring is 52px with a 4px stroke. Inside each ring, an emoji icon. Below each ring, the pillar name in the pillar's color.

The five pillars with their colors and icons:
- Community — #34737A (dark teal) — 🤝
- Confidence — #979DB6 (dusty lavender) — ✨
- Resilience — #D32027 (Saint John's Red) — 🛡️
- Readiness — #5DA592 (sage green) — 🎯
- Wellness — #EEBBA7 (peach) — 💚

Each ring fills to the user's progress percentage for that pillar.

---

## Data model

Add a new table: **user_progress**
- id
- user_id (foreign key → users.id)
- pillar (enum: community, confidence, resilience, readiness, wellness)
- progress (integer, 0-100)

One row per user per pillar. The overall percentage displayed in the large ring is the average of all five pillar values.

## Admin capability

In the Admin Panel > Users tab, when editing a user, add the ability to set progress values (0-100) for each of the five pillars. This is how staff update a client's progress manually.

---

## Visibility rules

- **Clients:** See their own progress dashboard.
- **Alumni:** See their completed dashboard (all rings full).
- **Staff and Admin:** Do not see the dashboard on their own Home page. They update client progress via Admin Panel > Users.
